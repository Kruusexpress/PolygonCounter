# Polygon POL Counter dApp

A professional Polygon smart contract dApp that allows users to:

- Increment counter
- Decrement counter
- Earn +100 reward tokens per click
- Connect wallet using MetaMask
- Interact with Polygon network
- Receive ERC20 rewards automatically
